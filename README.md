# Retail Sales Data Analysis

## Overview
This project performs end-to-end retail sales data analysis using Python and SQL. The objective is to clean raw transaction data, extract business insights, and visualize revenue trends.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (SQLite)
- Matplotlib
- VS Code

## Key Tasks Performed
- Data cleaning and preprocessing using Pandas
- Date transformation and monthly revenue aggregation
- SQL aggregation queries (GROUP BY, SUM, ORDER BY)
- Identification of top-performing products
- Category-wise sales performance analysis
- Revenue trend visualization using Matplotlib

## Business Insights Generated
- Monthly sales growth trends
- Top 5 revenue-generating products
- Category-level performance comparison
